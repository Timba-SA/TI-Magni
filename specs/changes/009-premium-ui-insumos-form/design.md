# Design: Premium UI for Insumos Form

## Architecture
- **Component**: `InsumoForm.tsx`
- **UI Framework**: React + Tailwind CSS + shadcn/ui (Dialog).

## UI Specifications
- **Background**: `bg-[#0a0a0a]/95` with `backdrop-blur-2xl`.
- **Accents**: `#FF5A00` for primary actions and highlights.
- **Borders**: `border-white/[0.07]` for that subtle glass edge.
- **Typography**: Mono fonts for labels to maintain the technical/editorial feel.
