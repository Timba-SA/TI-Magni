# Proposal: 000-project-baseline

## Contexto
El proyecto The Food Store ya cuenta con un desarrollo avanzado, incluyendo un frontend funcional en React y un backend en FastAPI con arquitectura por capas.

## Intención
Este baseline tiene como objetivo principal documentar el estado actual del proyecto para establecer una línea base de conocimiento.

## Alcance
- **Sí incluye:** Documentación de la arquitectura actual, stack tecnológico, dominios existentes, endpoints de la API y establecimiento de las reglas SDD.
- **NO incluye:** Introducción de nuevas funcionalidades, refactorización de código existente ni modificaciones a la infraestructura actual.

## Justificación
Al establecer este baseline, dejamos el repositorio preparado para trabajar con asistentes de IA (como Antigravity) utilizando la metodología Spec-Driven Development (SDD). Esto evita que futuras IAs asuman que el proyecto está vacío y proporciona un marco de trabajo seguro y estructurado.
