# Acceptance Criteria: 003-fullstack-insumos-crud

- El frontend permite listar los insumos leídos de la BD real.
- Un usuario puede agregar un nuevo insumo y este persiste en la BD.
- Un usuario puede editar nombre y stock de un insumo existente.
- Un usuario puede "eliminar" un insumo, el cual pasa a estado Inactivo (Baja Lógica).
- La UI refleja actualizaciones automáticamente tras las acciones.
