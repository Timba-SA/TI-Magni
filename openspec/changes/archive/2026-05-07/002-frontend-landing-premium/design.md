# Design: 002-frontend-landing-premium

## UI/UX
- **Tema:** Dark mode "glassmorphism", colores vibrantes en contraste con fondos negros.
- **Tipografía:** Moderna y editorial (Inter/Roboto).
- **Herramientas:** React Router para navegación fluida sin recargas. Tailwind CSS estricto, sin CSS puro adicional innecesario.

## Componentes
- `HeroSection`: Call to action principal con animaciones de entrada.
- `FeaturesGrid`: Muestra las capacidades del sistema (Inventario, Auth).
- `Navbar`: Transparente con blur (backdrop-filter) fijado arriba.
